INSTALLATION
-------------
1. download the files
2. put the files to a folder of your prefrenece
3. go with terminal the folder of your preference
4. write "npm install"
5. write "node app.js"
6. go to localhost:3000 or 127.0.0.1:3000
7. Hello! Yes, this is Dog!
